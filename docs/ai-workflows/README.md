# WePicTool AI 工作流

这个目录用于沉淀后续 AI 分类、抠图、白底合成、质量检测和提示词实验。当前阶段只保存工作流规格和提示词草案，不代表小程序已经调用真实 AI 服务。

## 当前文件

- `classification-prompt.md`：阶段二图片部件识别提示词和返回格式。
- `matting-prompt.md`：阶段三抠图提示词和调用方式（DashScope qwen-image-2.0）。
- `matting-api-evaluation.md`：阶段三抠图 API 选型与测试标准。
- `workflow-backlog.md`：后续 AI 工作流优先级和验收标准。
- `prompt-engineering-plan.md`：提示词工程化方案——抽取可复用 Block 与工作流封装（待实施）。

## 使用原则

1. 先完成阶段一真机验收，再接入真实 AI 分类。
2. 分类标签固定，不让模型自由发挥标签。
3. 试穿全身图默认 `daily`，不自动抠整个人做白底。
4. 头像、包、配饰、日常合集、表情包和心情状态图不进入 MVP 主链路。
5. 每次新增提示词或工作流，都要补充测试素材和验收标准。
