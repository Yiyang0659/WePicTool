# 抠图 API 选型评估

**阶段：** 阶段三  
**用途：** 选择用于穿搭白底卡片生成的抠图服务。  
**当前状态：** 已接入 DashScope `qwen-image-2.0` 做抠图，结果上传到云存储 `matted/` 目录。白底卡片合成（前端 Canvas）待实现。

## 候选服务

| 服务 | 状态 | 备注 |
| --- | --- | --- |
| **DashScope qwen-image-2.0** | **✅ 已接入** | 阿里云多模态生成模型，通过提示词去除背景替换纯白。边缘质量依赖生成模型能力，非专业分割。 |
| Remove.bg | 未测试 | 国际服务，效果稳定性需要结合网络和价格评估 |
| 稿定科技抠图 API | 未测试 | 国内图像服务候选 |
| 阿里云图像分割 | 未测试 | 国内云服务候选 |
| 腾讯云图像能力 | 未测试 | 国内云服务候选 |

## 测试素材

| 素材 | 数量 | 重点 |
| --- | --- | --- |
| 上衣 | 5 | 白色、黑色、条纹、蕾丝/毛边、宽松版型 |
| 下装 | 5 | 牛仔裤、黑裤、半裙、短裤、浅色裙 |
| 鞋子 | 5 | 运动鞋、高跟鞋、靴子、浅色鞋、复杂鞋带 |
| 非 MVP 素材 | 5 | 头像、包、帽子、项链、完整试穿图 |

## 对比维度

| 维度 | 要求 |
| --- | --- |
| 边缘干净程度 | 毛边、蕾丝、鞋带不能大面积残缺 |
| 浅色衣物可见性 | 白色衣服在白底上仍可辨认 |
| 单张耗时 | 平均 <= 5 秒 |
| 失败率 | 失败必须明确返回，不用脏结果冒充成功 |
| 价格 | 可支撑 MVP 内测成本 |
| 接入复杂度 | 适合 CloudBase 云函数或独立 Node 服务调用 |

## 阶段三生成要求

- 只对 `tops`、`bottoms`、`shoes` 进入抠图。
- 抠图在云函数中完成（DashScope qwen-image-2.0），结果上传云存储 `matted/` 目录。
- 白底卡片合成在前端 Canvas 完成（CloudBase 不支持 `sharp` 等原生模块）。
- 支持 1:1、4:5、3:4 三种比例。
- 主体最长边占画布 78%-86%。
- 上衣偏中上，下装偏中下，鞋子居中偏下。
- 白色/浅色衣物自动加轻阴影、细描边或浅灰底。
- 单张失败不影响整批结果展示。
